Web Technology and its applications

Course Outcomes:

The student will be able to :

  • Adapt HTML and CSS syntax and semantics to build web pages.
  • Construct and visually format tables and forms using HTML and CSS
  • Develop Client-Side Scripts using JavaScript and Server-Side Scripts using PHP to generate and display the contents dynamically.
  • Appraise the principles of object oriented development using PHP
  • Inspect JavaScript frameworks like jQuery and Backbone which facilitates developer to focus on core features.
